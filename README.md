# Images gallery

## Features
    - simple login without signup
    - images upload
    - adding comments to images
    - notifications when new comment appear

## Technology
    - HTML5
    - CSS3
    - SASS
    - JavaScript
    - Node.js
    - Socket.io
    - Gulp

## Dev run
<pre>
    <code>$ npm install</code>
</pre>

## Start application
<pre>
    <code>$ node app.js</code>
</pre>
